
# Laravel Livewire Starter

Livewire starter kit with authetication and user roles.

## Installation

- git clone https://github.com/isaacdarcilla/livewire-starter
- cd livewire-starter
- composer install
- npm install
- npm run prod
- php artisan storage:link
- php artisan migrate
- php artisan db:seed
- php artisan serve

Don't forget to set the `.env`
